Introduction

Este es un documento de ejemplo para ilustrar algunas de las caractersticas de IPython Notebooks. Un IPython notebook es otra herramienta de programacin literaria, similar a R Markdown. Sin embargo, en este caso est diseada para utilizarse con el lengauje de programacin Python.

IPython es un intrprete de Python que ofrece muchas caractersticas avanzadas, ponindo especial inters en facilitar la programacin interactiva y la comparticin rpida de informacin en programacin cientfica. En esta lnea, los notebooks proporcionan un mtodo muy sencillo y eficiente para crear y organizar documentacin de programacin cientfica con Python que, adems, permite editar y modificar los contenidos, as como evaluar los resultados de inmediato. Esto es posible gracias a que cada notebook cuenta con un intrprete de Python ejecutndose en segundo plano, que evalua las secciones de cdigo insertadas y devuelve los resultados. La interfaz es accesible desde cualquier navegador web, trabajando en modo local (normalmente http://localhost:8888).

Instalacin

Para instalar IPython e IPython notebook solo hay que seguir estas instrucciones. Si Python no est instalado por defecto en nuestro sistema operativo, la opcin ms recomendable es descargar e instalar directamente una distribucin para programacin cientfica en Python, como por ejemplo Anaconda.

Por otro lado, se pueden consultar tambin numerosos ejemplos de notebooks creados por otros usuarios, y disponibles en el proyecto nbviewer. Por ltimo, indicar que las partes de IPython independientes del lenguaje Python se han abstrado para crear el proyecto Jupyter, que permite la creacin de documentos de programacin interactiva utilizando mltiples lenguajes.

import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d.axes3d import Axes3D


def flux_qubit_potential(phi_m, phi_p):
    return 2 + alpha - 2 * cos(phi_p)*cos(phi_m) - alpha * cos(phi_ext - 2*phi_p)

alpha = 0.7
phi_ext = 2 * pi * 0.5
phi_m = linspace(0, 2*pi, 100)
phi_p = linspace(0, 2*pi, 100)
X,Y = meshgrid(phi_p, phi_m)
Z = flux_qubit_potential(X, Y).T

fig = plt.figure(figsize=(14,6))

# El argumento projection='3d' en add_subplot hace que el objeto reconozca un entorno 3D
ax = fig.add_subplot(1, 2, 1, projection='3d')

p = ax.plot_surface(X, Y, Z, rstride=4, cstride=4, linewidth=0)

# surface_plot con gradiente y barra en color
